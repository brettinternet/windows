## Windows Installation

Use <kbd>Shift</kbd> + <kbd>F10</kbd> for command prompt.

To setup Windows 11 without signing into Microsoft, enter command before signing into a network:

```
oobe\bypassnro
```

Open settings window:

```
start ms-settings:
```
